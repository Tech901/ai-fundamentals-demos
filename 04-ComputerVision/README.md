# MS Foundry: Computer Vision

This demo illustrates using Azure AI Computer Vision tasks like multi-class or multi-label image classification and object detection.

## Prerequisites

- Azure Subscription
- An Azure Resource Group + Workspace
- And endpoint and access key to the Azure Computer Vision service.

### Services

- [Azure Portal](https://portal.azure.com)
- [Vision Studio](https://portal.vision.cognitive.azure.com/gallery/face)
- [Microsoft Foundry](https://ai.azure.com/)
